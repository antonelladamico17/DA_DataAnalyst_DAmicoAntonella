Urban Sound Classification

×

The aim of this project is to create a Neural Network which is able to classify urban sound contained in UrbanSound8K dataset. Three different Neural Network architectures were developed with three different features extracted from audio files (MFCCs, Mel-Spectogram and Chromagram) as input.

The project explores various approaches to audio classification, comparing the effectiveness of different feature extraction methods and neural network architectures. This comprehensive study provides insights into the best practices for urban sound classification tasks.

Market Basket Analysis

×

Market Basket Analysis on IMDb dataset using Frequent Pattern Growth algorithm to find item associations. The goal is to find frequent itemsets - items that are more likely to appear together in the same baskets. In this project, movies are considered as baskets and actors as items.

This analysis helps identify patterns in movie casting, revealing which actors frequently appear together in films. The implementation uses PySpark for efficient processing of large datasets, making it suitable for big data applications.

Breast Cancer Risk Analysis

×

Statistical analysis of breast cancer risk factors using Mixed Interaction models on blood analysis data. The dataset contains records related to women patients located in Coimbra (Portugal) who have been diagnosed or not with breast cancer.

This study employs advanced statistical methods to identify significant risk factors and their interactions, providing valuable insights for medical professionals in understanding breast cancer risk assessment through blood analysis parameters.

Road Safety Dashboard

×

Interactive Shiny application for analyzing road accidents in the Lombardy region from 2010 to 2016. The application allows users to calculate statistics for injured and deceased people in specific districts or years, providing valuable insights for public safety decisions.

The dashboard features intuitive visualizations and filtering capabilities, enabling stakeholders to identify trends, high-risk areas, and temporal patterns in road accidents. This tool supports evidence-based decision making for road safety improvements.
